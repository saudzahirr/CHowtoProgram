The only identifiers with function-prototype scope are those
used in the parameter list of a function prototype. function
prototypes do not require names in the parameter listâ€”only
types are required. If a name is used in the parameter list
of a function prototype, the compiler ignores the name.
Identifiers used in a function prototype can be reused
elsewhere in the program without ambiguity.
